# **App Name**: Gemini Chat - Light

## Core Features:

- Phone Number Input: Allows users to enter their phone number for login.
- Password Input: Enables users to enter their password.
- Login Button: Submits the entered credentials for authentication.
- Forgot Password Link: Provides a link for users to reset their password.
- Sign Up Link: Directs new users to the registration page.
- Animated Input Fields: Subtle animation on input focus to enhance user experience.

## Style Guidelines:

- Primary color: Soft Blue (#B0E2FF) to evoke trust and calmness.
- Background color: Light Gray (#F5F5F5) for a clean, modern look.
- Accent color: Light Orange (#FFB347) for interactive elements and highlights, to complement the blue.
- Body and headline font: 'Inter' (sans-serif) for a clean and modern reading experience.
- Input fields highlight with a subtle color transition on focus.
- Use simple, outlined icons for input field labels (phone, password).
- Center-aligned form elements within a container with rounded corners for a softer appearance.
